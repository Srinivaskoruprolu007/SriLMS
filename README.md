# SriLMS Project Documentation

## Project Structure
- `src/`: Contains all the source code files.
- `tests/`: Includes test cases for the project.
- `docs/`: Documentation related files.

## Features
- User authentication and authorization.
- Course management system for admins and users.
- Interactive user interface for learning and engagement.

## Tech Stack
- **Frontend**: React.js
- **Backend**: Node.js with Express
- **Database**: MongoDB
- **Deployment**: Heroku

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Srinivaskoruprolu007/SriLMS.git
   ```
2. Navigate into the project directory:
   ```bash
   cd SriLMS
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
5. Visit `http://localhost:3000` in your web browser to view the application.
